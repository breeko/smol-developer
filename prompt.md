Python interactive tic-tac-toe game

Design and build an interactive rock-paper-scissors game

The user is prompted to input rock, paper or scissors, through r, p or s, respectively.
if the user puts in a nonsensical value, ask her again.

Once selected, the computer will randomly choose rock paper or scissors.

Rock beats scissors, paper beats rock, scissors beats paper.

The game keep tracks of your record against the computer and asks if you want to play again after every game.

Please don't include any external libraries and separate the model from the game logic in a separate file
